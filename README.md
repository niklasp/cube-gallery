# Cube Gallery

[![Dependabot badge](https://flat.badgen.net/dependabot/wbkd/webpack-starter?icon=dependabot)](https://dependabot.com/)

CSS 3d cube gallery for all your media 

### Installation

```
npm i --save cube-gallery
```
